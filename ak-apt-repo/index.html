🔐 AK API Key Manager - APT Repository

This is the APT repository for the AK API Key Manager. Choose your installation method:

📦 Quick Installation

curl -fsSL https://apertacodex.github.io/ak/setup-repository.sh | bash
sudo apt install ak

🔧 Manual Setup

# Add GPG key
curl -fsSL https://apertacodex.github.io/ak/ak-repository-key.gpg | sudo apt-key add -

# Add repository
echo "deb https://apertacodex.github.io/ak/ak-apt-repo stable main" | sudo tee /etc/apt/sources.list.d/ak.list

# Update and install
sudo apt update && sudo apt install ak

📂 Repository Structure

  • dists/ - Distribution metadata
  • pool/ - Package files
  • Release - Repository information
  • ak_2.1.0-1_amd64.deb - Latest package

✅ Verification

ak --version
ak --help
